性能监控

本插件使用了 stat.js 实现，会在指定位置显示性能信息

使用​

import {StatsSystem} from '@eva/plugin-stats'

const game = new Game({
  autoStart: true, // 可选
  frameRate: 60,
})

game.addSystem(new StatsSystem({
  show: true // 这里设置是否显示，设为 false 不会运行。
  style: { // 这里到数值全部都是宽度到百分比 vw 单位
    x: 0,
    y: 50,
    width: 20,
    height: 12
  }
}))
